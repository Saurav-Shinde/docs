---
title: "Make and view orders"
---

There is a **Make Order** button on dashboard.

Clicking it opens a window with three options:

1. Calculate Recipe Cost
2. View Dish Summary
3. Make Order

### Calculate Recipe Cost

Client selects a recipe from dropdown and system calculates production cost.

---

### Dish Summary

Shows cost summary of all available dishes.

---

### Place Order

Click **Make Order → Make Order again**

Now order window opens:

- Select Dish
- Quantity auto loads
- Price auto loads

Client can:

- Change quantity as per requirement
- Add multiple dishes

Then click **Place Order**

The order is sent to **Admin Order Management**

### Viewing Orders

Client can open **Orders tab** from dashboard.

Here client can:

- View placed orders
- Track status
- Mark order as **Received** after delivery
